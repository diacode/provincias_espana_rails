Provincias España Rails
========================================================================

Conjunto de archivos para integrarlos con un proyecto Rails ya existente con todas las provincias de España

Archivos incluidos
------------------------------

+	**Migrate**: db/_create_provincia.rb. Creación de la tabla
+ **Seeds**: db/seeds.rb. Con todas las provincias.
+ **Model**: models/provincia.rb. Modelo validando que el nombre de la provincia esté presente.
